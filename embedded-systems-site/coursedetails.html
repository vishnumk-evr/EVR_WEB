Technical Syllabus

Master embedded systems from the hardware level up. Each module combines theory with hands-on labs and real-world projects.

Duration: 2 weeks

Topics Covered:

  • 8-bit vs 32-bit microcontroller architectures (PIC, AVR, ARM Cortex)
  • Memory organization: ROM, RAM, EEPROM mapping and optimization
  • GPIO configuration, pin control, and digital I/O operations
  • Interrupt service routines (ISRs) and exception handling
  • Hardware debugging protocols: SWD, JTAG, Serial Wire
  • Low-power modes and power consumption optimization

Hands-On Labs: GPIO blink patterns, interrupt-driven LED control, timer configurations, low-power modes implementation

Hardware Used: PIC18F microcontroller board, ARM Cortex-M4 development kit, oscilloscope for signal analysis

Duration: 2 weeks

Topics Covered:

  • Pointers and pointer arithmetic in embedded contexts
  • Structures, unions, and bit-fields for memory-efficient data organization
  • Bit-wise operations: AND, OR, XOR, NOT, bit shifting for register manipulation
  • Memory management: malloc, free, stack vs heap allocation
  • Function pointers and callbacks for event-driven programming
  • Volatile keyword and compiler optimizations in embedded code
  • Code optimization for speed and memory constraints

Hands-On Labs: Pointer-based GPIO control, bit-field register access, memory profiling, callback implementations

Projects: Write a custom memory allocator, implement bit-wise flags system

Duration: 2.5 weeks

Topics Covered:

  • UART (Serial): Asynchronous communication, baud rates, frame formats, flow control
  • I2C (Two-Wire): Master-slave communication, addressing, clock stretching, multi-master scenarios
  • SPI (Three-Wire): Synchronous protocols, clock modes, chip select, full-duplex communication
  • CAN Bus: Automotive protocols, arbitration, message filtering, CAN FD
  • USB Basics: Device enumeration, endpoints, control/bulk/interrupt transfers (overview)
  • Protocol debugging and signal analysis with oscilloscope/logic analyzer

Hands-On Labs: UART echo test, I2C temperature sensor reading, SPI SD-card communication, CAN message transmission

Projects: Multi-device I2C network, SPI data logger with SD card, CAN vehicle telemetry simulator

Duration: 2.5 weeks

Topics Covered:

  • RTOS concepts: tasks, scheduling, preemption, time-slicing
  • FreeRTOS kernel: task creation, suspension, deletion, priorities
  • Context switching and interrupt handling in RTOS environment
  • Synchronization primitives: mutexes, semaphores, binary & counting semaphores
  • Inter-Process Communication (IPC): queues, message passing
  • Deadlock avoidance, priority inversion, and real-time guarantees
  • Task profiling and performance analysis

Hands-On Labs: FreeRTOS task scheduling, mutex-protected critical sections, producer-consumer queue patterns

Projects: Multi-tasking temperature monitor, real-time event-driven system, embedded server with multiple clients

Duration: 3 weeks

Topics Covered:

  • ESP32 & ESP8266: Wi-Fi/Bluetooth/BLE programming with Arduino IDE and FreeRTOS
  • Wireless Protocols: Wi-Fi basics, Bluetooth Low Energy (BLE) for wearables, mesh networking
  • Cloud Platforms: AWS IoT Core, Azure IoT Hub, Firebase Realtime Database, MQTT protocol
  • OTA Updates: Over-the-air firmware updates for deployed devices
  • Raspberry Pi Fundamentals: Linux basics, GPIO control via Python, interfacing with peripherals
  • Web server implementation on embedded devices (HTTP/HTTPS)
  • RESTful API design for IoT applications
  • Data logging and cloud integration patterns

Hands-On Labs: ESP32 Wi-Fi connection, BLE device pairing, MQTT pub-sub messaging, AWS IoT certificate setup

Projects: Smart environmental monitoring system, IoT weather station with cloud dashboard, Bluetooth smart home controller

Duration: 3 weeks

Topics Covered:

  • Schematic Capture: Component selection, ERC (Electrical Rules Check), netlist generation
  • PCB Layout (KiCad / EasyEDA): Routing, layer stackup, impedance control, ground planes
  • Design for Manufacturability (DFM): Trace widths, clearances, via placement
  • Gerber Generation & Fabrication: Panelization, picking the right fab service
  • Testing & Debugging: Multimeter usage, continuity checks, in-circuit programming
  • Capstone Project Design: Scoping a real embedded product, requirements, constraints
  • Integration & Deployment: Assembly, firmware loading, field testing

Hands-On Labs: Design a simple motor-driver PCB, route a microcontroller board, order and assemble a test PCB

Capstone Projects: Design a complete IoT product (hardware + firmware + cloud), e.g., smart home sensor node, automotive logger, wearable health monitor

Capstone & Project Examples

Real-world projects that build your portfolio and demonstrate industry-ready skills.

🌡️ Smart Environmental Monitor

Build an ESP32-based sensor node that monitors temperature, humidity, and air quality. Send data to AWS IoT Core, visualize on a real-time dashboard.

Skills: IoT, MQTT, AWS, Web Dashboard

🚗 CAN Bus Vehicle Logger

Design a CAN bus data logger that reads vehicle diagnostics (RPM, speed, fuel level). Log to SD card and upload to cloud for analysis.

Skills: CAN Bus, Automotive, Data Logging

💓 Wearable Health Monitor

Create a BLE wearable that monitors heart rate, steps, and sleep patterns. Send real-time data to a smartphone app.

Skills: BLE, Wearables, Low-Power Design

⚙️ Custom Motor Controller PCB

Design and fabricate a custom PCB that controls brushless DC motors using PWM. Implement real-time feedback loop with FreeRTOS.

Skills: PCB Design, RTOS, Motor Control

🏠 Smart Home Hub

Build a central hub that controls lights, fans, and security cameras via mobile app. Use multi-protocol integration (Wi-Fi, Zigbee).

Skills: IoT, Multi-Protocol, Home Automation

🤖 Autonomous Robot

Program a robot that navigates autonomously using ultrasonic sensors, avoids obstacles, and reports location via GPS-enabled connection.

Skills: Robotics, Real-Time Control, Sensors
